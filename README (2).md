### No.Cuenta: 315239674	


Aplicación basada en la técnica pomodoro para enfoque; la aplicación cosnta de las siguientes funcionalidades:

* Se implemento la interfaz de usuario. Para esta se implemento un diseño unicamente con los botones necesarios.
* Se añadió la funcionalidades para ajustar:
	+ Idioma o lenguaje en el que se muestra la aplicación
	+ Funcionalidad para cambiar a tema claro obscuro según la preferencia del usuario.
* Se implementarón 3 modos de temporizador, enfoque, descanso y descanso largo; y al finalizar cada uno de estos temporzadores se implemento la vibración como señal de termino.
* Se incluyen frases motivacionales.
* Cuenta con la finalidad de guardar las sesiones que han sido completadas.

* De las funcionalidades antes mencionadas se presento un reto principalmente al implementar la interfaz de usuario, dado apenas me estaba familiarizando con la herramienta.
Además de incluir el cambio de lengauje o modo obscuro o claro.

### Propuestas 
1. Para una segúnda versión podría incluirse el ajuste de los tiempos de los modos, que el usuario pueda seleccionar el tiempo pára cada uno.

2. Tambien sería una buena opción incluir que al abrir otra pantalla se siga corriendo el temporizador.

3. Como un ajuste, cambiaria un poco el diseño de colores elegidos en un principio para que estos vayan acorde a lo que quiere transmitir la apliciación, que ayuden a facilitar la concentración.
